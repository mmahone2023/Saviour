# Saviour

This is a project that lets you play interactive experiences across multiple environments.

## Quick Start
Use these commands to run the game locally
```sh
git clone https://github.com/mmahone2023/Saviour.git
cd Savior
npm install --include=dev
npm run dev
```
Then open [http://localhost:3000](http://localhost:3000) with your browser to play the game.

## Executive Summary 
A game that lifts you up, while lifting others up.

Quick Pitch: There will be one main protagonist, but the player will be able to choose from a variety of protagonists.  The protagonist explores diverse environments, completes meaningful acts of kindness, while their mood and abilities grow. Each time the protagonist succeeds in accomplishing a deed, the player earns hearts.  As the heart count goes up, an increase in mood is displayed on the screen.  The character is also able to move faster as their heart counter increases.  The player can also earn coins, by receiving tips for helping people.  Those coins can be used to buy accessories.  
 
What makes Saviour unique?  
Saviour will not involve competition with other players, which could potentially add stress to players.  The game is designed to promote relaxation, that feel-good effect and positive emotional engagement. 
Saviour will automatically have four different environments to choose to play, instead of having to progress through one environment, in order to get to another one.

2. General Information

Target Audience: Anyone, but especially players who wish to use gaming to promote mental wellness
Platform(s): PC and Mobile (iOS/Android)
Technology Stack: HTML, CSS, JavaScript, React, Phaser
Monetization: Free play
Inspirations: My personal experience of feeling better after helping someone 

3. Core Experience & Gameplay Loop
Core Loop: The player controls the protagonist, so that hearts are accumulated. 
	   The player always encounters someone or something in need of help.
	   The player will be able to use coins earned to buy things that will increase their likelihood of success, in helping people. 
 
Key Mechanics: The player will be able to fly, jump, interact with their hands, and fight. 
Story/Setting: There will be four different worlds, which will consist of the player occupying the air, wilderness, city and ocean.
Visual/Art Style: Ideally, all four worlds will consist of places that actually exist.

Goal: I am making this to take people away from their daily lives, while making them feel better.
Scope: Saviour is designed to be a finite game.
